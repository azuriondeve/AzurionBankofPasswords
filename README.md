# 🔐 Azurion Bank of Passwords

A secure, minimal, and self-hosted password manager built with Node.js and Express.

---

## 🚀 Features

- 🔒 AES-256 encrypted vault (local storage not online lol)
- 🧠 Master password authentication
- 🔑 Secure password generator (15–30 chars)
- 🌐 Simple web interface
- 🔄 Automatic update checker for fix bugs or other's things :)

---

## 🛠️ Tech Stack

- Node.js
- Express
- Crypto (built-in)
- UUID
